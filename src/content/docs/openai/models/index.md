description: 可用于示例和 FAQ 的 OpenAI 官方模型 ID。
---

本列表基于官方 [OpenAI latest model guide](https://developers.openai.com/api/docs/guides/latest-model)，每次公开更新模型前都应刷新。

| 模型 ID | 适用场景 |
| --- | --- |
| [`gpt-5.5`](./gpt-5-5/) | 需要当前默认模型处理复杂工作流、推理型支持回答、工具型 agent 或长上下文审阅。 |
| `gpt-5.4-mini` | 需要较低延迟或成本，处理较轻量支持和路由任务。 |
| `gpt-5.4-nano` | 需要最小模型处理简单分类、格式化或短文本抽取。 |

## 命名规则

公开文档默认使用 OpenAI 官方模型 ID。不要在示例中添加服务名称、路由后缀或实验后缀。

## 发布清单

- 从 OpenAI 官方文档确认模型 ID。
- 将模型 ID 加入本表。
- 新增或更新专属模型 FAQ 页面。
- 使用最新模型运行 curl 示例检查。
