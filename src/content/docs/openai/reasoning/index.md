推理模型适合需要多步分析、仔细比较或先规划再回答的任务。[OpenAI reasoning guide](https://developers.openai.com/api/docs/guides/reasoning) 是 API 行为的主要参考。

本章节将官方说明整理成可复制的支持文档：

- [Reasoning effort](/openai/reasoning/effort/) 说明如何选择 effort。
- [Tokens and context](/openai/reasoning/tokens-and-context/) 说明输出预算。
- [Reasoning summaries](/openai/reasoning/summaries/) 说明哪些内容适合展示给用户。
- [Prompting](/openai/reasoning/prompting/) 给出 GPT 工作流的提示词结构。
- [Tool use](/openai/reasoning/tool-use/) 覆盖多轮和工具型工作流。
- [Examples](/openai/reasoning/examples/) 提供 curl 片段。
- [FAQ](/openai/reasoning/faq/) 汇总技术支持常用短答。
